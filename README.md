## Warning
smorth is just a small project I like to work on in my free time, and is very buggy at the moment.

## Compilation instructions
first bootstrap the buildsystem by compiling nob.c \
from there you can just run nob to build the project \
feel free to cange nob.c as much as you want, running nob will automatically recompile and run it

## Usage
you can run smorth with -st to display the stack as an expression is interpreted
